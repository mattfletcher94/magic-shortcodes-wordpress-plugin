== Installation ==

1. Install and active the plugin.
2. Navigate to the 'Magic Shortcodes' tab.
3. Click the 'Create Popup' button.
4. On this page, you can customize your popup.
5. Click the 'Save Popup' button when you've finished.
6. It should now appear on the front-end of your website.

== Frequently Asked Questions ==

= How many popups can I create? =

You can create an unlimited amount of popups.

= Can I add shortcodes to my popup? =

Yes.

= Can I limit the amount of times a user sees a popup? =

You can choose how often a user sees each popup. The options include:
Every page load, Once per session, Once per day and Once per week.

= Can I limit which pages the popup is shown on? =

You can display the popup on all pages, or manually select the pages to show it on.
